  • init(scriptName: string, scriptRaw: any[], parseCols: string[], objectifyCols: string[], processText?: string[]): void
    • Defined in Game/Core/Engine/LevelManager.ts:38
    description

    initialize the level manager.

    Parameters

    • scriptName: string

      the name of the script to initialize the script handler with

    • scriptRaw: any[]

      the raw script data for the script handler

    • parseCols: string[]

      the names of the columns to be parsed into arrays of names (i.e 'horse,dog,cat' => [horse, dog, cat])

    • objectifyCols: string[]

      the names of the columns to be converted into objects with key-value pairs. For example: 'bgd: bgd_1\noverlay: overlay_1' => {bgd: 'bgd_1', overlay: 'overlay_1'}

    • Optional processText: string[]

      (optional) the column names to convert into lines and words of text. Mainly useful in passage (reading) types.

    Returns void
